第一章：安装前准备

1.1 需要什么环境

项目要求说明
操作系统Windows 10/11家庭版或专业版均可
Python3.8 及以上（推荐 3.11）Required. 附下载链接
无线网卡任意 Windows 支持的无线网卡USB/PCIe/内置均可
管理员权限推荐用于禁用USB节能、重启网卡
PowerShell系统自带无需额外安装
netsh系统自带无需额外安装

1.2 下载 Python

推荐版本：Python 3.11.9（64位）

官方下载地址（直接点击下载）：

https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe

备选版本：Python 3.10.11（64位）

https://www.python.org/ftp/python/3.10.11/python-3.10.11-amd64.exe

Python 官网（选择其他版本）：

https://www.python.org/downloads/

下载 Python 3.11.9 (64位)

1.3 安装 Python（关键步骤，务必按顺序操作）

警告：以下步骤必须严格按顺序执行，否则软件无法启动！
  1. 双击运行下载的 python-3.11.9-amd64.exe
  2. ★★★ 最重要的一步 ★★★
    在安装界面最底部，勾选「Add Python 3.11 to PATH」
    如果不勾选，后面软件会启动不了！
  3. 点击「Customize installation」自定义安装
  4. 在 Optional Features 页面，确保勾选了：
    ✅ tcl/tk and IDLE（必须勾选，软件界面需要它）
    ✅ pip
    其他默认勾选即可
  5. 点击 Next，在 Advanced Options 页面：
    ✅ 勾选「Add Python to environment variables」
    建议安装路径改为：C:\Python311\（不要放中文路径）
  6. 点击 Install 开始安装
  7. 等待安装完成，出现「Setup was successful」表示成功
  8. 点击 Close 关闭

1.4 验证 Python 安装成功

  1. 按键盘 Win + R，输入 cmd，回车
  2. 在命令行输入：python --version
  3. 如果显示「Python 3.11.9」说明安装成功
  4. 再输入：python -c "import tkinter"
  5. 如果没有任何报错（直接跳到下一行），说明 tkinter 也安装成功了

1.5 关于额外依赖

好消息：本软件无需安装任何额外依赖！

本软件仅使用 Python 内置库（tkinter、subprocess、threading、json、os 等），不需要 pip install 任何第三方库。

第三章：启动软件

3.1 方式一：双击启动（推荐）

  1. 进入解压目录
  2. 双击「启动_WiFi监控.bat」
  3. 会弹出软件界面窗口

3.2 方式二：后台静默启动

  1. 双击「后台静默启动.vbs」
  2. 软件在后台运行，不显示窗口
  3. 如需显示界面，用方式一

3.3 方式三：命令行启动

  1. 按 Win + R，输入 cmd，回车
  2. 用 cd 命令进入解压目录：
cd /d C:\WiFi监控工具
  1. 输入启动命令：
python wifi_monitor.py

3.4 启动失败怎么办？

  1. 双击「启动_WiFi监控_诊断版.bat」
  2. 诊断脚本会自动检查：
    • Python 是否安装
    • tkinter 模块是否可用
    • wifi_monitor.py 文件是否存在
    • 代码语法是否有错误
  3. 根据提示修复对应问题

第四章：首次使用配置

4.1 以管理员身份运行（推荐）

重要：建议首次使用时以管理员身份运行，原因：
· 禁用 USB 节能功能需要管理员权限
· 重启网卡设备需要管理员权限
· 静默重装驱动需要管理员权限

操作方法：

  1. 右键点击「启动_WiFi监控.bat」
  2. 选择「以管理员身份运行」

4.2 基本设置

在软件界面的设置区域进行配置：

设置项默认值建议说明
Target SSID空（自动检测）保持默认留空自动连接当前网络
检测间隔5秒保持默认多久检测一次WiFi状态
最大重试次数10次保持默认超过后自动停止
重连间隔3秒保持默认重连失败后等待时间
启动时自动开始监控不勾选可勾选开机即自动监控
禁用 USB 节能勾选强烈推荐防止USB网卡省电断连
失败N次后重启网卡3次保持默认触发网卡设备重启

配置完成后点击「保存设置」

4.3 开始监控

  1. 点击蓝色的「开始监控」按钮
  2. 状态面板会实时显示当前 Wi-Fi 信息
  3. 日志区会显示检测和重连记录

4.4 验证功能是否正常

  1. 确认软件显示「监控状态：运行中」
  2. 确认状态面板能看到当前 Wi-Fi 名称、信号强度等
  3. 手动断开 Wi-Fi 测试：
    • 关闭 Wi-Fi 开关
    • 等待5-10秒
    • 观察日志是否显示「检测到断开」并自动重连
    • 确认 Wi-Fi 恢复连接

第五章：日常使用

5.1 界面说明

状态面板（实时刷新）：

  • 监控状态：运行中/已暂停/已停止
  • 当前 Wi-Fi：当前连接的WiFi名称
  • 重试次数：当前重试 / 累计重连
  • 信号强度：百分比
  • 信道：当前信道号
  • 传输速率：Mbps
  • IPv4：当前IP地址
  • 网关 / 延迟：网关IP及ping延迟(ms)

操作按钮：

  • 开始监控（蓝色）：启动监控
  • 暂停（黄色）：暂停监控（不退出）
  • 停止（红色）：完全停止监控
  • 保存设置：保存配置
  • 状态诊断（橙色）：一键诊断WiFi状态
  • 清空日志：清空日志区
  • 更新日志（标题栏橙色按钮）：查看版本更新记录

5.2 设置开机自启

  1. 按 Win + R，输入：shell:startup，回车，会打开启动文件夹
  2. 把「后台静默启动.vbs」的快捷方式复制到这个文件夹
  3. 以后开机会自动运行

5.3 更新软件

  1. 下载新的 win10_wifi_auto_reconnect.zip
  2. 把 zip 文件放到软件目录下
  3. 双击「一键更新.bat」
  4. 脚本会自动解压并覆盖旧文件

第六章：高级配置

6.1 配置自动重装驱动

适用场景：网卡经常彻底挂掉，重启网卡也救不回来

第1步：录制驱动安装响应文件

  1. 准备你的网卡驱动安装程序（.exe 文件）
  2. 按 Win + R，输入 cmd，回车
  3. 执行录制命令（把路径换成你的驱动路径）：
"C:\Drivers\Setup_MW.exe" /r /f1"C:\WiFi监控工具\setup.iss"
  1. 会弹出驱动安装界面，按正常流程安装一遍
  2. 安装完成后，setup.iss 文件已生成

第2步：在软件中配置

  1. 打开软件设置面板
  2. 驱动安装程序路径：填入 C:\Drivers\Setup_MW.exe
  3. 静默响应文件路径：填入 C:\WiFi监控工具\setup.iss
  4. 保存设置

配置后，软件连续失败7次以上会自动静默重装驱动。

6.2 禁用 USB 节能的原理

Windows 默认开启 USB 选择性挂起功能，会在空闲时断开 USB 设备供电以省电。这对于 USB 无线网卡是致命的——网卡被断电后连接直接断开。本软件启动时会自动修改注册表禁用此功能，需要管理员权限。

第七章：常见问题 FAQ

Q1：双击 bat 文件闪退怎么办？
说明 Python 没装好或 PATH 没配。用「启动_WiFi监控_诊断版.bat」排查，或重新安装 Python 务必勾选「Add Python to PATH」。
Q2：提示"ModuleNotFoundError: No module named 'tkinter'"怎么办？
tkinter 没安装。重新安装 Python，在「Customize installation」步骤勾选「tcl/tk and IDLE」。
Q3：软件能打开但没有 Wi-Fi 信息显示？
检查电脑是否有无线网卡，确认 Wi-Fi 开关已打开，确认 WLAN AutoConfig 服务正在运行（services.msc 查找 WLANSvc）。
Q4：软件显示"未找到无线网卡"？
1. 确认无线网卡已插入（USB 网卡检查接口）
2. 按 Win + R 输入 ncpa.cpl 检查是否有无线网络连接
3. 如果网卡被禁用，右键启用
Q5：禁用 USB 节能需要管理员权限，不勾选可以吗？
可以，软件能正常运行。但如果你用的是 USB 无线网卡，强烈建议以管理员身份运行并勾选，能大幅减少断连。
Q6：软件支持什么网卡？
支持所有 Windows 能识别的无线网卡（USB/PCIe/内置均可），已在水星 MW150US USB 网卡上测试通过。
Q7：日志在哪里看？
软件界面下半部分就是实时日志区。日志不会写入文件，关闭软件后日志清空。
Q8：连续失败10次自动停止后怎么办？
点击「停止」再点「开始监控」即可重新开始。建议用「状态诊断」按钮排查问题。
Q9：手动关闭电脑 WLAN 开关后软件能自动重连吗？
可以。v2.3版本已支持检测 WLAN 无线电关闭状态，会自动尝试 netsh autoconfig + 重启网卡恢复无线电，然后继续重连。
Q10：软件会自动重装驱动吗？
默认不会。需要在设置中配置驱动安装程序路径和静默响应文件路径后，连续失败7次以上才会触发自动重装。
Q11：必须用管理员权限吗？
不是必须。但以下功能需要管理员权限：禁用USB节能、重启网卡设备、静默重装驱动。普通权限下软件仍可正常监控和重连，只是恢复手段少一些。
Q12：支持 Windows 7 吗？
不推荐。软件使用 PowerShell 5.0+ 命令，Windows 7 自带的 PowerShell 版本较低可能不兼容。建议 Windows 10 及以上。

第九章：注意事项

  1. Python 安装时务必勾选「Add Python to PATH」，这是最常见的启动失败原因
  2. Python 安装时务必勾选「tcl/tk and IDLE」，否则软件界面无法显示
  3. USB 无线网卡用户强烈建议以管理员身份运行并勾选「禁用 USB 节能」
  4. 软件路径可以包含中文，但建议路径不要有空格
  5. 不要删除解压目录中的任何文件，特别是 wifi_monitor.py
  6. 更新软件时用「一键更新.bat」，不要手动覆盖文件
  7. 驱动重装功能为可选高级功能，不配置不影响正常使用
  8. 如遇无法解决的问题，使用「状态诊断」按钮一键诊断
